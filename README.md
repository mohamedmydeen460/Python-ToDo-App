# To-Do List Application (Python)

## Description
A console-based To-Do List application developed using Python.
It allows users to add, view, and delete daily tasks.

## Features
- Add new tasks
- View existing tasks
- Delete tasks
- Menu-driven program

## Technologies Used
- Python

## How to Run
1. Install Python
2. Open terminal / command prompt
3. Run:
   python todo.py

## Concepts Used
- Lists
- Loops
- Menu-driven logic
- Indexing
